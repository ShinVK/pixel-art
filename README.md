## Pixel Art

Esse projeto foi construído em javascript, html e css.
O objetivo do projeto é criar um quadro de pixels que permite ser pintado e criar pixel arts.

---

## Aprendizados

É um projeto feito no início do aprendizado de programação.
Com o intuito de aprimorar o uso de funções, assim como a lógica de programação.
A construção dos elementos html de forma semântica, para que seja bem organizada e de acordo com os padrões.
